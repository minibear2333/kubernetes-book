system命令

用法：

docker system COMMAND

四个参数

  • df：docker磁盘使用率
  • events：和docker events用法一样
  • info：和docker infos一样
  • prune：删除没有在使用的资源，谨慎使用

https://docs.docker.com/engine/reference/commandline/system/

docker system df
docker system df -v
docker system events
docker system info
docker system prune
